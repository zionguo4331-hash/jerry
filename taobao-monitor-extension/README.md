# 淘宝竞品监控 Chrome 扩展

## 功能特点

- **自动监控**: 浏览淘宝/天猫商品详情页时自动提取数据
- **变动检测**: 自动对比历史数据，检测价格、主图、销量变动
- **一键导出**: 生成 Markdown 报告，自动保存到指定目录
- **数据存储**: 本地存储，支持导出 JSON 备份

## 安装步骤

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `taobao-monitor-extension` 文件夹

## 配置 Chrome 默认下载目录

为实现自动保存到 Obsidian，需要设置 Chrome 默认下载目录：

### 步骤 1: 创建 Obsidian 目录
```
~/Documents/GitHub/jerry/obsidian/
```

### 步骤 2: 配置 Chrome
1. 打开 Chrome 设置 (`chrome://settings/`)
2. 找到"下载内容"部分
3. 点击"更改"按钮
4. 选择 `~/Documents/GitHub/jerry/obsidian/` 目录
5. 关闭"下载前询问每个文件的保存位置"

### 步骤 3: 重新加载扩展
1. 访问 `chrome://extensions/`
2. 点击扩展的"重新加载"按钮

## 使用方法

### 自动监控
1. 正常浏览淘宝/天猫商品详情页
2. 扩展会自动提取：标题、价格、销量、主图、评价、问大家
3. 数据自动保存到浏览器存储

### 查看报告
1. 点击浏览器工具栏中的扩展图标
2. 查看今日报告和变动统计

### 导出报告
- **导出 MD**: 生成 Markdown 格式竞品日报，自动保存到默认下载目录
- **导出 JSON**: 导出完整数据备份

## 数据说明

### 提取的信息
| 字段 | 说明 |
|------|------|
| title | 商品标题 |
| price | 价格 |
| sales | 月销量 |
| mainImage | 主图 URL |
| reviews | 评价列表 |
| questions | 问大家 |
| shopName | 店铺名称 |

### 变动检测
- 价格变动
- 销量变动
- 主图变动
- 评价更新

## 目录结构

```
obsidian/
├── 竞品分析/       # 存放各商品详细分析
└── 竞品日报/      # 存放每日汇总报告
```

## 注意事项

1. 数据存储在浏览器本地，清除浏览器数据会丢失
2. 建议定期导出 JSON 备份
3. 如果导出未自动保存到目标目录，请检查 Chrome 下载设置
4. 如遇验证码，请手动完成验证后继续浏览

## 故障排除

### 数据未提取
- 确保浏览的是商品详情页（URL 包含 item.htm 或 detail.tmall.com）
- 尝试刷新页面后等待 2-3 秒
- 检查扩展是否已启用

### 导出失败
- 确认 Chrome 下载目录已正确设置
- 检查是否有弹出窗口被阻止
